# React Monolab

A React UI library.

## 📦 Installation

Run the following command:

```bash
npm install --save react-mono
```

## 🔨 Usage

```js
import { Button, DatePicker } from "react-mono";

const App = () => (
  <>
    <Button type="primary">PRESS ME</Button>
    <DatePicker />
  </>
);
```
